# ezClaps
Ubuntu 20.04 Package Installer for Incident Response

## Installation
1. Install [Ubuntu 20.04](https://releases.ubuntu.com/)
2. Run the following commands in Terminal
```bash
cd /tmp
wget https://codeload.github.com/0xShen/ezClaps/zip/master
unzip master
chmod +x setup.sh
./setup.sh
```
